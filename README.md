# Graphs

This project has three data structures: 
- **Singly linked list**;
- **Doubly linked list**;
- **Binary tree**.

 Each of the entities is template and can work with 5 data types:
- **int**;
- **double**;
- **string**;
- **char**;
- **bool**.
